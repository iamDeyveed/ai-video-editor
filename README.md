# ClipAI

ClipAI is a browser-based video editor built with Next.js. It supports AI caption generation, live styling, trimming, and browser-native export recording.

## Environment

Create `.env.local` with:

```bash
OPENAI_API_KEY=your_openai_api_key_here
```

## Local development

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.
